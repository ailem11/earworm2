# Earworm2

A full-stack application for identifying the song that's in your head.

## Architecture
- **Backend:** Python (FastAPI/Flask) located in `/backend`. Uses a virtual environment.
- **Frontend:** Vanilla HTML/JS located in `/frontend`.

## Setup Instructions
1. **Backend:** - `cd backend && source venv/bin/activate`
   - `pip install -r requirements.txt`
   - `python app.py`
2. **Frontend:**
   - Open `/frontend/index.html` in a browser.
